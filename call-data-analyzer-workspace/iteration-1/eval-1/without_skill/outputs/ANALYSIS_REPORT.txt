CRITICAL FINDINGS - WHY SENTIMENT IS NEGATIVE
================================================================================

PRIMARY FRICTION POINTS (across negative interactions):

1. Bot Inability to Categorize (especially "Other" category)
   - 61.8% of "Other" issues abandoned by users
   - Bot lacks context to understand ambiguous requests
   - Users frustrated when bot keeps asking clarifying questions
   Impact: Drives highest negative sentiment (64.6%)

2. Unclear Information from Bot
   - Refund timelines not clearly communicated
   - Shipping tracking status not updated proactively
   - Bot repeats generic responses instead of specific information
   Impact: Frustration escalation, repeat interactions

3. Resolution Process Complexity
   - Return/exchange process not intuitive via bot
   - Return options not easily discoverable in app/website
   - Bot transfers needed before customer satisfaction achieved
   Impact: 59.3% escalation rate on returns/refunds

4. Communication Gaps with Third Parties
   - Delivery partner information not integrated into bot
   - Refund processing status not tracked/communicated
   - Bot cannot provide revised timelines
   Impact: 50.5% negative sentiment on shipping delays

5. Bot Limitations in Understanding Context
   - Failed to understand customer language/variations
   - Unable to grasp multi-step issues
   - Transfer required for policy exceptions
   Impact: 53.3% escalation for order issues

================================================================================
METHODOLOGY & DATA LIMITATIONS
================================================================================

Methodology:
1. Loaded 5,386 voice bot call records with sentiment labels
2. Consolidated 47 granular issue categories into 6 consolidated types
3. Calculated sentiment distribution for each category
4. Analyzed CSAT scores, call duration, and resolution patterns
5. Identified friction points from text summaries and friction_points field

Data Quality Notes:
- Missing data: 3,621 records lack issue_category/resolution_status (67%)
- CSAT scoring incomplete: Only 1,727 records (32%) have CSAT scores
- Bot effectiveness field: categorical ("Low", "Medium", "High")
- Sentiment field: Pre-labeled ("Positive", "Neutral", "Negative")

Limitations Without Specialized Skill:
- Manual text parsing of friction points (sampling approach)
- Excel file generated with pandas/openpyxl (no advanced analytics)
- Limited NLU analysis without ML tools
- No trend analysis over time (no timestamp in data)
- No clustering of similar friction points
- All calculations performed in Python without database optimization
